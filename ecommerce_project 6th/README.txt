E-Biz 
Includes: index.php, products.php, cart.php, register.php, login.php, logout.php, admin.php (simple), db.php, styles.css, init.sql
Instructions:
1. Create a MySQL database (e.g., ebiz_demo).
2. Import init.sql to create tables and sample data.
3. Put files in a PHP-enabled server (e.g., XAMPP, LAMP) document root.
4. Update db.php with DB credentials.
5. Access index.php in your browser.
